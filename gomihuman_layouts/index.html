📍 画面遷移ガイド

🌐 共通画面の流れ

01. トップページ → 各ロールのログイン画面（02/03/04）または新規登録（05）へ遷移
05. 新規会員登録 → 登録完了後、02. クライアントログインへ遷移

🏢 クライアント画面の流れ

02. クライアントログイン → ログイン成功後、06. クライアントダッシュボードへ遷移
06. クライアントダッシュボード → 各機能画面（07〜15）へのハブとして機能
　├ 07. サービス依頼作成 → 依頼送信後、06へ戻る
　├ 08. 企画・見積確認 → 承認後、09. 契約締結へ遷移
　├ 09. 契約締結・素材提出 → 完了後、10. 運営計画確認へ遷移
　├ 10. 運営計画確認 → 確認後、11. サービス実施状況へ遷移可能
　├ 11. サービス実施状況 → リアルタイム監視画面（自動更新）
　├ 12. 収集データ閲覧 → データ確認後、13. 実施報告書へ遷移可能
　├ 13. 実施報告書閲覧 → 報告書確認後、14. 請求書確認へ遷移可能
　├ 14. 請求書確認・支払 → 支払完了後、15. フィードバックへ遷移
　└ 15. フィードバック入力 → 送信後、06へ戻る

⚙️ 管理者画面の流れ

03. 管理者ログイン → 2FA認証後、16. 管理者ダッシュボードへ遷移
16. 管理者ダッシュボード → 各管理機能（17〜26）へのハブとして機能
　├ 17. 会員情報管理 → 会員の追加/編集/削除後、16へ戻る
　├ 18. サービス依頼一覧 → 依頼選択後、19. 企画・見積作成へ遷移
　├ 19. 企画・見積作成 → 作成後、18へ戻るか20へ遷移
　├ 20. 契約・素材管理 → 確認後、21. 運営計画作成へ遷移
　├ 21. 運営計画作成 → 作成後、22. スタッフ・資材手配へ遷移
　├ 22. スタッフ・資材手配 → 手配完了後、23. サービス実施モニタへ遷移
　├ 23. サービス実施モニタ → 監視中、24. データ集計へ遷移可能
　├ 24. データ集計・分析 → 分析後、25. 実施報告書作成へ遷移
　├ 25. 実施報告書作成 → 作成後、26. 請求書発行へ遷移
　└ 26. 請求書発行 → 発行後、16へ戻る

👤 スタッフ画面の流れ

04. スタッフログイン → ログイン成功後、27. スタッフダッシュボードへ遷移
27. スタッフダッシュボード → 各作業機能（28〜33）へのハブとして機能
　├ 28. 作業指示一覧 → 作業選択後、29. 作業詳細閲覧へ遷移
　├ 29. 作業詳細閲覧 → 確認後、30. 活動記録入力へ遷移
　├ 30. 活動記録入力 → 記録後、31. ゴミ処理入力へ遷移
　├ 31. ゴミ処理入力 → 入力後、32. 作業完了報告へ遷移
　├ 32. 作業完了報告 → 報告完了後、27へ戻る
　├ 33. 勤怠管理 → 出退勤記録後、27へ戻る
　│
　├─ ⚡ クイックアクション（27から直接アクセス可能）
　　├ 34. 活動開始 → 準備チェック完了後、29. 作業詳細へ遷移
　　├ 35. 緊急連絡 → 緊急時の連絡・報告（どの画面からもアクセス可）
　　└ 36. マニュアル確認 → 必要な情報を確認後、元の画面へ戻る

💡 ヒント：各画面の番号は、システム全体の処理フローを表しています。
基本的に番号順に進むことで、サービスの一連の流れを完了できます。
